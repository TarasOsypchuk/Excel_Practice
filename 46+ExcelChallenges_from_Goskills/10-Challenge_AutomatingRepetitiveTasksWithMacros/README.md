# \# Excel Challenge #10: Automating Repetitive Tasks With Macros

# 

# This repository contains my solution to the Excel Challenge #10 from GoSkills. This challenge focuses on process automation using Excel Macros (VBA) to streamline quarterly web analytics reporting and standardize visual data representations.

# 

# \## 📋 Task Overview

# 

# !\[Excel Challenge 10 Task](./10-Challenge\_img/Task-file-1.png)

# 

# 

# The project handles recurring web analytics traffic metrics generated every three months. To eliminate manual, repetitive restructuring tasks across multiple quarterly sheets, the objective is to build a one-click automation pipeline that cleans the layout and injects specific visual tracking models.

# 

# \### 🎯 Key Objectives:

# 1\. \*\*Traffic Breakdown Visualization:\*\* Generate a tailored chart for each worksheet demonstrating new vs. returning visits segment distributions grouped by country.

# 2\. \*\*Account Proportion Analysis:\*\* Generate a secondary chart explicitly showing the percentage or relative proportion of paid accounts across different countries.

# 3\. \*\*Chart Type Optimization:\*\* Evaluate the source data layouts to autonomously select the most appropriate chart types (e.g., Clustered Column vs. Pie/Donut) for volume comparisons and distribution metrics.

# 4\. \*\*Macro Recording \& Automation:\*\* Pack the entire generation, positioning, and design workflow into a recorded macro executable across multiple sheets with a single click.

# 5\. \*\*Standardized Layout Design:\*\* Ensure all generated charts embed clean formatting masks, proper descriptive labels, and structured reference titles positioned natively alongside the raw table grids.

# 

# \---

# 

# \## 🛠️ Data Engineering \& Analysis Steps

# 

# \* \*\*Macro Scripting:\*\* Recorded and calibrated a reusable macro script designed to run seamlessly across independent multi-sheet quarterly datasets.

# \* \*\*Analytical Representation:\*\* Evaluated data distribution attributes to deploy column series for traffic volume segmentation and relative percentage charts for subscriber account ratios.

# \* \*\*Component Standardization:\*\* Enforced absolute layout policies to inject dynamic chart titles, clean axis lines, and matching color schemes automatically.

# \* \*\*Cross-Sheet Scalability:\*\* Hardcoded structural offset targets within the routine to guarantee clean chart rendering positions without overlapping underlying raw data blocks.
